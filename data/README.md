# Dataset Folder

Place the larger SEER/Kaggle Breast Cancer dataset CSV file in this folder.

Required filename:

```text
seer_breast_cancer.csv
```

The dataset CSV is not uploaded automatically because Kaggle datasets may require manual download through a Kaggle account.

After placing the CSV file here, run:

```bash
python train_model.py
```

The script will generate updated figures and artifact files automatically.
